# 🖱️ Click Counter

A beginner-friendly JavaScript project built with **HTML**, **CSS**, and **JavaScript**.

This project was created to practice JavaScript fundamentals such as DOM manipulation, event handling, functions, variables, and conditional statements.

## ✨ Features

- ➕ Increase counter
- ➖ Decrease counter
- 🔄 Reset counter
- 📢 Display status messages
- 🚫 Maximum and minimum count limits
- 🎨 Interactive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 What I Learned

While building this project, I practiced:

- Selecting HTML elements with `querySelector()`
- Handling button clicks with `addEventListener()`
- Updating the DOM using `textContent`
- Working with variables and functions
- Using `if...else` conditions
- Organizing JavaScript into reusable functions
